## Особенности

- Процедурная генерация мира (в том числе с помощью шума перлина)
- Подземелья: 4 этажа, процедурная расстановка комнат с туманом войны
- 15 типов врагов (4 этажа сложности)
- 4 босса с 3 фазами каждый
- Режим арены (режим бесконечной башни с волнами врагов)
- Физика pymunk (отбрасывание, коллизии)
- Локализация: русский / английский
- Рывок с неуязвимостью
- карта подземелья
- Анимированный лут с перспективой (OpenCV)
- Система сохранений

## Управление

| Клавиша        | Действие                      |
|----------------|-------------------------------|
| WASD / Стрелки | Передвижение                  |
| SPACE          | Рывок (неуязвимость)          |
| J / ЛКМ        | Атака                         |
| E              | Взаимодействие (сундуки, NPC) |
| M              | карта (подземелье)            |
| ESC            | Пауза / Выход                 |
| O              | Сменить язык                  |
| P              | Сменить шрифт                 |
| F11            | Полноэкранный режим           |
| + / -          | Громкость SFX                 |

## Установка и запуск

```bash
git clone <repo-url>
cd sonetaiko
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python main.py
```
